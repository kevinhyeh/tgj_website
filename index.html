How to Create Your Project Easily

Create a project with TGJ's free project builder in 5 easy steps.

1. Starting a project with us is easy. Just click on "Start Design" up top to begin.

2. We have several projects for you to pick from. If there's something you need but we don't have, just choose the "other" option.

3. Customize your project by choosing how complex you'd like it as well as the color scheme if you have one. Don't worry, it's optional.

4. Add some information about the project and yourself. We'd like to know as much information as we can to give you the best possible experience.

5. Once you submit then you're done. You can also add more projects if needed. Some of your information will still be there so you don't have to re-type them.
